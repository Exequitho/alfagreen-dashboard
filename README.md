### **🛠️ Technical Architecture & Data Quality Workflow**
Rather than simply dropping incomplete records, I prioritized data retention and statistical validity to ensure accurate forecasting across all 311 Indonesian regencies/cities.

* **Data Engineering & Cleansing:** Handled extensive missing values (nulls) in the national dataset using Python (`Pandas`). Instead of executing `.dropna()`, which would have skewed regional representation, I imputed missing values using the statistical mean of each specific waste category.
* **Exploratory Data Analysis (EDA):** Conducted univariate and bivariate analysis to identify geographic hotspots, pinpointing DKI Jakarta as the highest daily waste producer and Papua Barat Daya as the lowest.
* **Predictive Modeling:** Engineered a Linear Regression model (`y = 1.512 * x - 129.941`)  to prove a strong positive correlation between general daily waste volume and inorganic waste accumulation. 
* **BI Visualization:** Designed an interactive executive dashboard in Google Looker Studio, featuring correlation scatter plots, waste distribution maps, and year-over-year trend lines (2019-2022).

### **📈 Actionable Recommendations for Stakeholders**

Based on the data modeling and dashboard analytics, I delivered the following strategic recommendations to the Product and Engineering teams:

1. **Targeted Phased Rollout:** Initiate the Alfagreen logistics and collection network exclusively in DKI Jakarta. Expanding nationwide immediately would cause severe logistical bloat; Jakarta provides the highest density of recoverable inorganic waste to stress-test the system.
2. **Incentivize High-Yield Materials:** The imputed data revealed specific distributions of plastics, metals, and glass. The UI/UX team should design the Alfagreen reward system to offer higher "Alfagift Points" for the specific inorganic materials that yield the highest resale value to third-party recyclers.
3. **Continuous Model Training:** The current linear regression model accurately maps the 2019-2022 trends. Once Alfagreen goes live, the data pipeline must be automated to feed real-time user recycling data back into the model to predict future logistics capacity needs.
